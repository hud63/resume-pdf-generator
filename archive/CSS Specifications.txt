## Typography Specifications

- **Primary Font**: Inter, weight range 300-700
- **Fallback**: Helvetica Neue, Arial, sans-serif
- **Main Name**: 28pt, weight 700, letter-spacing -0.5pt
- **Section Headers**: 10pt, weight 700, uppercase, letter-spacing 0.8pt
- **Sidebar Headers**: 9pt, weight 600, lowercase, letter-spacing 0.3pt
- **Body Text**: 9pt, weight 400, line-height 1.5
- **Small Text**: 7-8pt, line-height 1.3-1.4

## Spacing System

- **Page margins**: 15mm top/bottom, 12mm left/right
- **Column gap**: 8mm
- **Section spacing**: 6-8mm between sections
- **Paragraph spacing**: 2-3mm between elements
- **Line spacing**: 1.3-1.5x font size
